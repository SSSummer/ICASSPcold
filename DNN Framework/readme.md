# DNN-for-ICASSP

## Environment

- Python 3.5
- Tensorflow 1.3.0
- keras 2.0.5
- ubuntu 16.04

### Instruction 
There are 3 files for our DNN experiment.

- DNN.ipynb                        to get DNN model
- DNNtranser.ipynb                 to get DNN-Cold model
